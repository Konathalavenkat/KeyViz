Test your keystroke display extension

📋 How to Test

  1. Make sure the Keystroke Display extension is installed and enabled
  2. Click the extension icon and toggle "Enable Display" to ON
  3. Start typing in the fields below or anywhere on this page
  4. Watch your keystrokes appear in the bottom-right corner!

🎯 Try These Shortcuts

  • Enter - New line
  • Backspace - Delete
  • Shift + Any Key - Uppercase/Symbols
